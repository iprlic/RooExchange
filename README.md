RooExchange
===========
Example dApp -> Roo Token Exchange

### Commands:
```
  Compile:              truffle compile
  Migrate:              truffle migrate
  Test contracts:       truffle test
  Test dapp:            npm test 
  or:					yarn test
  Run dev server:       npm run start
  or:					yarn start
  Build for production: npm run build
  or:					yarn build
```
